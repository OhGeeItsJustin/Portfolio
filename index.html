Public Art Hamilton

Visit Website Application Video View Source Code
 

Description
Public Art Hamilton is a website that uses open data from Open Hamilton about public art locations around the city of Hamilton. The website displays those locations and the users GPS location on a Google map as markers. When the user selects a marker more information about the selected location will appear such as the name, description, address, and city in the markers label.

Project Difficulties
This project was my first time working with Bootstrap to try and make a responsive web design for my website so it would look good on desktop and mobile devices.

My Solution
I spent two weeks going through the documentation for Bootstrap learning from the examples. After I felt more comfortable with the examples I would copy them to a HTML file and start tweaking the examples. From tweaking the documentation examples I discovered styling options that I could apply to the Public Art Hamilton website.

Technologies Used

  • HTML5/CSS3
  • PHP
  • Bootstrap
  • gmaps.js

More Mohawk

Application Video View Source Code
 

Description
More Mohawk was my final capstone project that was an Android application that was targeted towards to first year students Mohawk College students. The application only included information first year Mohawk students would care about such as important campus locations and information about clubs on campus.

Project Difficulties
More Mohawk had user accounts and clubs involved with the application. The application needed the ability to store data.

My Solution
I spent time researching ways to store for my Android application. At first I was going to use a SQLite database but released this would take a large amount of time to create an application that would store the data locally on the phone then upload the data after the application was closed because all of the club information was allowed to be accessed by all student accounts. Instead I integrated Cloud Firestore to retrieve and send all the data that was being used in my application.

Technologies Used

  • Java
  • Cloud Firestore
  • Maps SDK for Android
